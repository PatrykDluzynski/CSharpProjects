The pong clone was the first game I made in the unity game engine. It's simple with the only controls being W and S to move up and down, and ESC to close the game down. The mouse also has to be used to click the play button. Try it out!

The Zig Zag clone was my second practice project. It has a more friendly menu at the start. The game has no sound as I was focusing more on understanding how to correctly implement movement and understanding how physics are handled. Z is used to change direction, and X is used to jump. ESC pauses the game, and a help marker can be toggled on and off in the options menu, to show where tha ball will land after jumping. Try it out!

The Fruit Ninja clone was the last project I attempted before taking the CS50 course. All the assets apart from the sound were made by me. The game is played using the mouse to guide the blade. Each fruit gives a different amount of points, and bombs cause the player to lose a life. After losing 3 lives the game is over. Pressing ESC will pause the game, allowing to resume, restart, access options or quit. Try it out!
